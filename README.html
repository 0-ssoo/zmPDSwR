Example code and data for "Practical Data Science with R" by Nina Zumel and John Mount, Manning 2014.

The book:

  • "Practical Data Science with R" by Nina Zumel and John Mount, Manning 2014

Zip file of code excerpts from the book:

Example data sets:

Includes works derived from others (data sets) remain controlled by them. We are distributing as these parties have allowed and not making any claim or grant of additional rights or license.

  • Bookdata Data about book ratings.
  • Buzz Data about forum activity.
  • CDC Data about US CDC birth statistics.
  • Custdata Synthetic example data derived from Census PUMS data to demonstrate retail related plots.
  • KDD2009 Data about credit account prediction.
  • PUMS Data from US Census PUMS.
  • Protein Data about dietary protein sources across multiple countries
  • SQLExample Synthetic example data relating price to hotel reservation pickup.
  • Spambase Data about a email spam classification.
  • Statlog Data about German loans
  • UCICar Data about synthetic car ratings.
